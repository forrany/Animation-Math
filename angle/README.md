# 🎮 角之勇者大冒险 - 数学学习游戏

一个专为小学生设计的卡通风格角色扮演类数学学习游戏，将角的度量知识点融入有趣的冒险旅程中。

## 🌟 游戏特色

- **🎯 教育性**：完全贴合小学四年级数学课程标准
- **🎨 视觉性**：卡通风格界面，生动有趣的图形绘制
- **🎮 游戏性**：RPG升级系统、装备收集、战斗机制
- **📱 兼容性**：支持电脑和移动设备，响应式设计
- **💾 便捷性**：自动保存游戏进度，可随时继续

## 🎯 学习目标

通过游戏化学习，学生将掌握：
- 直线、射线、线段的概念和区别
- 角的分类（锐角、直角、钝角、平角、周角）
- 角的度量（1平角=180°，1周角=360°）
- 角度的计算和应用
- 钟表上的角度问题
- 几何图形中的角

## 🎲 游戏玩法

### 1. 创建角色
- 输入勇者名字开始冒险
- 自动创建角色档案

### 2. 探索地图
- 8个主题区域逐步解锁
- 20个关卡循序渐进
- 每个区域对应不同的知识点

### 3. 战斗系统
- **回合制问答**：答对题目对怪物造成伤害
- **血量机制**：知识力（HP）和计算力（MP）
- **敌人反击**：答错会被攻击
- **击败怪物**：通过答题击败怪物完成关卡

### 4. 角色成长
- **经验系统**：答对题目获得经验值
- **升级机制**：经验满自动升级，提升血量和魔法值
- **装备收集**：完成任务获得装备，增加准确率
- **技能解锁**：等级提升解锁新技能

### 5. 奖励机制
- **即时奖励**：答对获得经验和金币
- **装备奖励**：通关获得特殊装备
- **成就系统**：连续答对、完美通关等成就

## 📱 使用说明

### 如何开始游戏
1. 打开 `index.html` 文件（推荐使用Chrome、Firefox等现代浏览器）
2. 输入勇者名字
3. 点击"开始冒险"
4. 从第一关开始挑战

### 如何进行游戏
1. **选择关卡**：在地图上点击解锁的关卡
2. **阅读题目**：仔细阅读题目和图形
3. **选择答案**：点击选项或输入答案
4. **战斗胜利**：击败怪物完成关卡
5. **获得奖励**：领取经验、金币和装备
6. **继续冒险**：解锁下一关卡

### 保存与进度
- 游戏进度自动保存到浏览器本地存储
- 关闭浏览器后重新打开可点击"继续游戏"
- 支持7天内的存档恢复

### 移动设备
- 完全支持触屏操作
- 响应式界面自动适配屏幕大小
- 推荐横屏游戏以获得更好体验

## 🔧 开发者说明

### 文件结构
```
├── index.html          # 主HTML文件
├── style.css           # 样式文件
├── game.js            # 游戏逻辑
└── README.md          # 说明文档
```

### 代码架构
- **Game类**：主游戏控制器
- **levelsData**：关卡数据定义
- **Canvas绘图**：几何图形绘制函数
- **本地存储**：saveGame/loadGame方法

### 自定义关卡
如需添加新关卡，在 `game.js` 的 `levelsData` 数组中添加：
```javascript
{
    id: 21,
    title: "新关卡名称",
    type: "choice", // 或 "input"
    enemy: { name: "敌人名", hp: 血量 },
    question: {
        stem: "题目内容",
        options: ["选项1", "选项2", "选项3", "选项4"], // 选择题需要
        correct: 正确答案索引
    },
    reward: { exp: 经验, coins: 金币, item: 装备 }
}
```
